# 📸 Instagram Caption Generator (Python)

A fun and interactive **Python CLI project** that generates stylish Instagram captions based on your mood, vibe, or situation. This project is built using **if-else and nested if-else**, making it perfect for beginners learning Python control flow.

---

## 🔗 Live Repository

👉 https://github.com/rishii-in/Instagram_caption_generator_python.git

---

## 🚀 Features

* 😊 Mood-based caption generator
* 🌦️ Weather-based captions (sunny / rainy)
* 🙏 Tradition captions (temple / home)
* 🚗 Ride captions
* 🍜 Food captions
* 🔁 Continuous loop (use multiple times)
* ❌ Exit option
* ⚠️ Error handling for invalid inputs

---

## 🧠 Concepts Used

This project helps you practice:

* `if`, `elif`, `else` statements
* Nested `if-else`
* `while` loops
* User input handling
* Exception handling (`try-except`)
* String formatting

---

## 🖥️ How It Works

1. The program displays a menu with different mood options.
2. The user selects a choice (1–6).
3. Based on the selection:

   * A relevant Instagram caption is generated.
   * Some options ask for extra input (weather/place).
4. The program runs continuously until the user exits.
